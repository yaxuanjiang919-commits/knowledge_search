# README.md
## 本地知识库搜索系统
基于 Rust 开发的命令行文档检索工具，实现文档加载、倒排索引构建、关键词检索功能，可快速检索本地 Txt / Markdown 文档。

## 一、项目简介
本项目为 Rust 课程期末作业，采用模块化设计，运用 Rust 所有权、借用、枚举、结构体、特征、自定义错误处理等核心特性。
程序支持读取本地目录/单个文档，自动构建并持久化**倒排索引**，通过关键词快速匹配相关文档，是简易搜索引擎的工程实践。

## 二、运行环境与依赖
### 1. 基础环境
- 操作系统：Windows / Linux / macOS
- Rust 工具链：`rustc` + `cargo`（推荐最新稳定版）

### 2. 第三方依赖
项目使用以下开源 crate，配置在 `Cargo.toml` 中，编译时自动拉取：
1. `clap`：命令行参数解析
2. `serde` + `serde_json`：数据序列化与反序列化，实现索引文件持久化

## 四、编译与运行
### 1. 编译项目
在项目根目录打开终端，执行编译命令：
```bash
cargo build
```

### 2. 运行程序
本程序通过**命令行参数**控制功能，支持三类参数：
- `-p / --path`：【必填】指定文档路径（文件/文件夹）
- `-k / --keyword`：【选填】检索关键词
- `-r / --rebuild`：【选填】强制重新构建索引

#### 用法1：仅加载文档，构建索引（首次运行）
```bash
cargo run -- -p ./docs
```

#### 用法2：加载索引 + 关键词搜索
```bash
cargo run -- -p ./docs -k Rust
```

#### 用法3：强制重建索引并执行搜索
```bash
cargo run -- -p ./docs -r -k 索引
```

#### 用法4：读取单个文档并搜索
```bash
cargo run -- -p ./docs/doc1.txt -k 所有权
```

## 六、核心功能说明
1. **文档加载**：自动识别 `.txt`、`.md` 格式文档，支持单文件与整个目录读取；
2. **倒排索引**：对文档内容分词，构建关键词与文档的映射关系；
3. **索引持久化**：索引自动保存为 `index.json`，下次运行可直接加载，无需重复构建；
4. **关键词检索**：根据输入关键词快速查询，格式化展示所有匹配文档路径；
5. **参数校验**：拦截空关键词、非法路径等异常输入，错误提示清晰。

## 七、技术亮点（课程作业专用）
1. **模块化编程**：功能拆分多个独立模块，代码解耦、结构清晰；
2. **完善错误处理**：自定义错误枚举，全程使用 `Result` + `?` 传递错误，未使用 `unwrap`；
3. **所有权与借用**：大量使用引用传递数据，避免内存拷贝，遵循 Rust 内存安全规则；
4. **特征（Trait）抽象**：自定义特征实现通用行为，结合序列化特征完成数据持久化。

## 八、现存不足与优化方向
1. 当前仅按空白字符分词，**中文分词效果一般**，可引入专业中文分词库优化；
2. 暂不支持多关键词组合检索、结果关键词高亮；
3. 可补充单元测试，进一步提升代码健壮性。
